Rare-Variant Association Methods

Burden tests:

Strengths – are more powerful when there is a large proportion of variants are casual and effects are in the same direction

Limitations – less powerful in the presence of both trait increasing and decreasing variants or a small proportion of causal variants are often

  • CMC

  1. Idea: CMC method, also known as Combined Multivariate and Collapsing Method, uses allele frequencies to determine the divisions of the variants into groups. After the rare are selected, they are collapsed into an indicator variable and then a multivariate test is applied to the collection formed by the common variants and the collapsed variant.

  2. Model: For the CMC method, markers are divided into subgroups. (e.g. allele frequencies), and within each group, marker data are collapsed. A multivariate test (e.g. Hotelling’s T2 test) is then applied for analysis of the groups of marker data.

  3. R function

Single-marker test equation:

\[X_i^2 = N\frac{[p_i^D(aa) - p_i^N(aa)]^2}{p_i^D(aa)+p_i^N(aa)} + \frac{[p_i^D(Aa)+p_i^D(AA) - p_i^N(As) - p_i^N(AA)]^2}{p_i^D(Aa) +p_i^D(AA) + p_i^N(Aa) + p_i^N(AA)}\]

Multiple-marker test equation:

\[S = \frac{1}{N_A + N_A - 2}{{\sum_{j = 1}^{N_A}}{(x_j - \bar{x})(x_j - \bar{x})^T} + {\sum_{j = 1}^{N_\bar{A}}{(Y_j - \bar{Y})(Y_j - \bar{Y})^T}}} \]

Hotelling’s T^2 test:

\[T^2 = \frac{N_AN_\bar{A}}{N_A + N_\bar{A}}(\bar{X} - \bar{Y})^TS^{-1}(\bar{X} - \bar{Y})\]

Y: Numeric vector with phenotype status: 0=controls, 1=cases. No missing data allowed.

X: numeric matrix or data frame with genotype data coded as 0, 1, 2. Missing data is allowed.

Adaptive Burden tests:

Strengths – are stronger than burden tests using fixed weights or thresholds. Some tests can improve result interpretation

Limitations – are usually computationally intensive. VT requires the same assumptions as burden tests

  • aSum test

  1. Idea: The aSum test is also called an Adaptive sum test and it is the approach behind the adaptive test is to use the U-statistics of the score test in order to construct a statistic with the first components of the score vector U.

  2. Model: The Sum test can be used to summarize information on multiple Nucleotide Polymorphisms (SNP’s) with a DF=1. Adaptive Sum tests are low power and have limited control over error type I.

  3. R function

\[aSum=(U-U_0)'V_0^{-1}(U-U_0)\]

ASUM(y, X, perm = 100)

Variance-Component tests:

Strengths – are powerful in the presence of both trait increasing and decreasing variants or a small amount of causal variants

Limitations – are less powerful than burden tests when most variants are causal and effects are in the same direction

  • SKAT

  1. Idea: SKAT is a Sequencing Kernal Association Test which can be used to calculate p-values by fitting the null mode containing only the covariates and so it can easily be applied to genome-wide data.

  2. Model: SKAT tests have a relationship between variants and the phenotype by using classical multiple linear and logistic regression.

  3. R function

\[Q_{SKAT}=\sum_{j=1}^{m}w_j^2S_j^2\]

  • C-alpha test

  1. Idea: C-alpha tests allow for consistent and/or increased power and straightforward calculations when testing the variance of a set containing both protective and risk variants.

  2. Model: C-alpha is used to find patterns across a whole set of rare variants in a certain region. The distribution of counts follows a binomial distribution if alleles do not associate with any phenotype in the targeted region.

  3. R function

\[T=\sum_{i=1}^{m}[(y_i - n_ip_0)^2 - n_ip_0(1-p_0)]\]

\[c=\sum_{n=2}^{max n}m(n)\sum_{u=0}^{n}[(u-np_0)^2- np_0(1-p_0)]^2f(u|n,p_0)\]

Combined tests:

Strengths – are stronger with respect to the percentage of causal variants and the presence of both trait increasing and decreasing variants.

Limitations – can be slightly less powerful than burden or variance-component tests if their assumptions are largely held; some methods are computationally intensive like the Fisher method

  • SKAT-O

  1. Idea: SKAT-O is part of the extended family of SKAT and it is optimal unified test. This test doesn’t need to be resampled when computing asymptotic p values and it is easy to adjust to covariates.

  2. Model: The Qoptimal test, also know as the optimal unified test can be calculated to find the p-value by simply using a one-dimensional numerical integration. When the trait is binary, the sample sizes are small and the p-values have produced more conservative results, then this can lead to incorrect type I error and a loss of power.

  3. R function

\[Q_{optimal}=min_{0<p<1}p_p\]

\[Q_{optimal}=min[p_{p_1},...,p_{p_b}]\]

EC test:

Strengths – is powerful when a very small proportion of variants are causal

Limitations – is computationally intensive; is less powerful when a moderate or large proportion of variants are causal

  • EC test

  1. Idea: The EC test, standing for exponential combination test, is general rather than a one test statistic and it combines individual variant statistics leading to the making of a set-based analysis. This test leads to powerful combination procedures.

  2. Model: Large exponential weights may lead to low power when distinguishing the alternative from the null. When the alternative is less sparse, then it becomes a little more powerful.

  3. R function

\[Q_{EC}=\sum_{j=1}^{m}exp(\frac{S_j^2}{2var(Sj)})\]
